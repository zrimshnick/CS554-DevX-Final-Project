**TO RUN MAIN SERVER:**
from main directory "CS554-DevX-Final-Project":
1. cd Server
2. npm i
3. npm start

starts server on localhost:3000

---------------------------------
**TO RUN CHAT SERVER:**
from main directory "CS554-DevX-Final-Project":
1. cd Server-Chat
2. npm i
3. npm start

starts server on localhost:4000

--------------------------------
**TO RUN CLIENT SIDE:**
from main directory "CS554-DevX-Final-Project":
1. cd DevX-Project
2. npm i
3. npm run dev

starts webpage on localhost:5173
