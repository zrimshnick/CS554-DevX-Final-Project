---------------------------------
**TO RUN MAIN SERVER:**

1. cd Server
2. npm i
3. npm start

starts server on localhost:3000

---------------------------------
**TO RUN CHAT SERVER:**

cd Server-Chat
npm i
npm start

starts server on localhost:4000

--------------------------------
**TO RUN CLIENT SIDE:**

cd DevX-Project
npm i
npm run dev

starts webpage on localhost:5173
